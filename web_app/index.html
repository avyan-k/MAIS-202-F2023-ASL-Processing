American Sign Language Translator


How to use the ASL Translator

  1. To start the webcam, click the 'Start Webcam' button.
  2. Align your hand to the desired ASL hand sign and click 'Take Picture' to take the picture and have it translated.
  3. A notification for download will pop up. Save it in the folder called 'images' for the program to interpret.
  4. The corresponding letter should pop up on the line below. Repeat step 2 for each letter that needs to be translated.
  5. Once you are done, hit the 'Stop Webcam' button to end the webcam.
